## 📝 解题步骤

### 1. 分析问题

- 理解问题的目标

- 确定每一步的"最优"标准

### 2. 设计贪心策略

- 确定每一步如何选择
- 考虑边界情况
- 验证策略的合理性

### 3. 证明贪心策略

- **贪心选择性质**：证明局部最优可以导致全局最优
- **最优子结构**：证明问题具有最优子结构
- **反证法**：假设存在更优解，推导矛盾

### 4. 实现算法

- 根据策略编写代码
- 处理边界情况
- 优化时间和空间复杂度

## 🔍 常见错误

### 错误1：没有排序

```javascript
// ❌ 错误：没有排序
function greedy(nums) {
    for (let num of nums) {
        // 处理逻辑
    }
}

// ✅ 正确：先排序
function greedy(nums) {
    nums.sort((a, b) => a - b);
    for (let num of nums) {
        // 处理逻辑
    }
}
```

### 错误2：贪心策略错误

```javascript
// ❌ 错误：每次都选择最大的
function greedy(nums) {
    let result = [];
    while (nums.length > 0) {
        let max = Math.max(...nums);
        result.push(max);
        // ...
    }
}

// ✅ 正确：根据问题设计合适的策略
function greedy(nums) {
    nums.sort((a, b) => a - b);
    // 根据具体问题设计策略
}
```

### 错误3：没有处理边界情况

```javascript
// ❌ 错误：没有处理空数组
function greedy(nums) {
    let sum = 0;
    for (let num of nums) {
        sum += num;
    }
    return sum;
}

// ✅ 正确：处理边界情况
function greedy(nums) {
    if (nums.length === 0) return 0;
    let sum = 0;
    for (let num of nums) {
        sum += num;
    }
    return sum;
}
```

## 💡 优化技巧

### 1. 排序预处理

大多数贪心问题都需要先排序：
- 区间问题：按开始时间或结束时间排序
- 分配问题：按大小排序
- 其他问题：根据问题特点排序

### 2. 双指针技巧

在排序后，使用双指针可以高效处理：
- 分发饼干：一个指针遍历孩子，一个指针遍历饼干
- 合并区间：一个指针遍历区间，维护前一个区间

### 3. 维护最值

很多问题需要维护最值：
- 跳跃游戏：维护最远可达位置
- 买卖股票：维护历史最低价
- 其他问题：根据问题维护相应的最值

## 📊 复杂度分析

### 时间复杂度

- **排序**：O(n log n)
- **遍历**：O(n)
- **总时间**：通常 O(n log n)

### 空间复杂度

- **排序**：O(1) 或 O(n)（取决于是否允许修改原数组）
- **其他**：通常 O(1)
- **总空间**：通常 O(1) 或 O(n)
