
给定一个二叉树的根节点 root ，返回 它的 中序 遍历 。

示例 1：

![二叉树中序遍历](../assert/inorder_1.jpg)

输入：root = [1,null,2,3]
输出：[1,3,2]
示例 2：

输入：root = []
输出：[]
示例 3：

输入：root = [1]
输出：[1]

```javascript
// 定义中序遍历函数
var inorderTraversal = function (root) {
    // 初始化结果数组，用于存储遍历结果
    const res = [];

    // 定义递归的中序遍历函数
    const inorder = (node) => {
        // 如果当前节点为空，直接返回
        if (!node) return;

        // 递归遍历左子树
        inorder(node.left);

        // 访问当前节点，将当前节点的值添加到结果数组中
        res.push(node.val);

        // 递归遍历右子树
        inorder(node.right);
    }

    // 从根节点开始进行中序遍历
    inorder(root);

    // 返回结果数组
    return res;
}
```

### 详细说明

1. **函数定义**：
   - `inorderTraversal` 是主函数，接收一个二叉树的根节点 `root` 作为参数。

2. **初始化结果数组**：
   - `const res = [];`：初始化一个空数组 `res`，用于存储中序遍历的结果。

3. **定义递归的中序遍历函数**：
   - `const inorder = (node) => { ... }`：定义一个递归函数 `inorder`，用于执行中序遍历。
   - `if (!node) return;`：如果当前节点为空，直接返回。这是递归的终止条件。
   - `inorder(node.left);`：递归遍历左子树。
   - `res.push(node.val);`：访问当前节点，将当前节点的值添加到结果数组 `res` 中。
   - `inorder(node.right);`：递归遍历右子树。

4. **从根节点开始进行中序遍历**：
   - `inorder(root);`：从根节点开始调用递归函数 `inorder`，进行中序遍历。

5. **返回结果数组**：
   - `return res;`：返回结果数组 `res`，其中包含中序遍历的节点值。

### 中序遍历的顺序

中序遍历的顺序是：

1. 先遍历左子树。
2. 访问当前节点。
3. 再遍历右子树。
