白屏时间（White Screen Time）是指用户打开一个网页或应用直到页面上首次出现任何可识别内容的时间。这段时间内，用户看到的是一片空白，因此称为“白屏时间”。白屏时间的长短直接影响到用户体验，过长的白屏时间会导致用户感到不耐烦，甚至放弃访问。

计算白屏时间的方法通常有两种：

### 1. 使用浏览器开发者工具

大多数现代浏览器（如Chrome、Firefox、Safari等）都内置了开发者工具，可以通过这些工具来测量白屏时间。

#### Chrome DevTools

1. **打开开发者工具**：
   - 按 `F12` 或 `Ctrl+Shift+I`（Windows/Linux）/ `Cmd+Opt+I`（Mac）。
2. **选择“Performance”标签**。
3. **点击“Record”按钮**（圆形图标）开始记录。
4. **在新标签页中打开你要测试的网页**。
5. **当页面加载完成后，点击“Stop”按钮**（方形图标）停止记录。
6. **查看记录的数据**：
   - 在“Performance”面板中，找到“Main”线程下的“First Contentful Paint”（FCP）时间点，这是白屏时间的结束点。
   - 从记录开始到FCP的时间就是白屏时间。

### 2. 使用自动化测试工具

自动化测试工具可以更精确地测量白屏时间，并且可以进行批量测试。常见的自动化测试工具包括Lighthouse、WebPageTest等。

#### Lighthouse

1. **打开Chrome浏览器**。
2. **打开你要测试的网页**。
3. **按 `F12` 打开开发者工具**。
4. **选择“Lighthouse”标签**。
5. **点击“Generate report”按钮**。
6. **在生成的报告中，查看“Performance”部分的“First Contentful Paint”（FCP）时间**，这就是白屏时间。

### 3. 使用JavaScript代码

你也可以在页面中嵌入JavaScript代码来测量白屏时间。

```html
<!DOCTYPE html>
<html>
<head>
    <title>白屏时间测试</title>
    <script>
        let startTime = performance.now();

        document.addEventListener('DOMContentLoaded', () => {
            let endTime = performance.now();
            let whiteScreenTime = endTime - startTime;
            console.log('白屏时间: ' + whiteScreenTime + ' ms');
        });
    </script>
</head>
<body>
    <h1>欢迎访问</h1>
</body>
</html>
```

在这个示例中，`startTime` 记录了页面开始加载的时间，`endTime` 记录了 `DOMContentLoaded` 事件触发的时间，两者的差值就是白屏时间。

### 总结

白屏时间的测量方法有多种，选择哪种方法取决于你的具体需求和环境。使用浏览器开发者工具和自动化测试工具可以提供更详细的性能数据，而嵌入JavaScript代码则适用于需要在生产环境中实时监控白屏时间的场景。
