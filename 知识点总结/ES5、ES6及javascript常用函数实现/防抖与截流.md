### 函数防抖实现
每次点击都会清除上一个定时器 函数 ，重新生成一个新的定时器函数 ，一段时间后执行

```
    function debounc(func,dely){
        let timer = null
        return function () {
            if(timer) clearTimeout(timer)
            timer = setTimeout(()=>{
                func()
            },dely)
        }
    }
```
